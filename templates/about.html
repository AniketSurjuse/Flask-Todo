About ToDo APP

Overview

Your TODO App is a simple and effective tool to help you manage your tasks and stay organized.

Features

  • Easy task management with a user-friendly interface.
  • Add, edit, and delete tasks effortlessly.
  • Search functionality to quickly find specific tasks.

Technologies Used

Your TODO App is built using the following technologies:

  • Flask - A lightweight web framework for Python.
  • SQLAlchemy - SQL toolkit and Object-Relational Mapping (ORM) for Python.
  • SQLite - A self-contained, serverless, and zero-configuration database engine.
